# Projeto de Quality Assurance - Residência Deloitte

Bem-vindo ao repositório oficial do nosso squad para o projeto prático da **Residência Tecnológica em Quality Assurance**, promovida pela Deloitte em parceria com o Porto Digital.

## Status Atual do Projeto
**Fase Inicial:** Estamos na etapa de planeamento e estruturação da documentação.

O nosso foco atual é a criação do **Plano de Testes** para o sistema **ParaBank**, um banco online de demonstração focado em aprendizado e validação de cenários de testes de software.

## Estrutura do Repositório (Até ao momento)
- `Plano de Teste/`: Diretório destinado à documentação central do nosso planeamento.
  - `versao1.0.md`: Arquivo principal contendo as especificações, estratégias e critérios de aceite do sistema.
